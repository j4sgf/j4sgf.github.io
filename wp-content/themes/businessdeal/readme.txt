== Description ==

BusinessDeal theme is an amazing modern Corporate and Business WordPress theme. This theme is suitable to build Business, Corporate Office, Business landing page, Personal business, Agency, Company, Finance agency and many more. It is easy to customize. It also include all major aspects like responsive, performance, cross-browser compatible, SEO ready and supports RTL. It is ready to promotion with social media icons to reach maximum target audience. Responsive slider impress your customers with lively eye-catching images right on your banner section. We provide the best support to you. Get free Supports at https://themespiral.com/forums, View demos at https://demo.themespiral.com/businessdeal and documentation at https://docs.themespiral.com/businessdeal

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= How to install Dummy Content ? =
 Please install One Click Demo Import plugins. You can install it after activating businessdeal theme. It is listed in recommended Plugins
 After plugin is activated, it asks you to upload  XML, WIE and  DAT dummy file
 Download it from https://themespiral.com/download/1248/
 Unzip businessdeal-dummy-content.zip file. You can find all XML, WIE and  DAT dummy file
 Upload manually and Click on Import demo data.

 Now all your files and settings has been imported. Now you just need to setup your menu and front page.

 Setup Menu:
 In the Blog Dashboard, select Appearance > Menus.
 Under the Menu Settings, located at the bottom of your screen, select Primary/ Secondary menu
 Click save menu
 Screenshot: https://docs.themespiral.com/wp-content/uploads/2019/02/primary-menu.png

 Setup Home Page:
 	- How to setup Home Template?
		Log in to your WordPress Administration Panel (Dashboard).

		Go To Dashboard > Pages > Add New and Give it a title.

		In the page editor, there is a box on the right called “Page Attributes“. From this box,select "Home Template" and save the page.

		Now your Home Template is setup. To setup Banner Read About Main Banner Documentation


	- How to setup Advertise Area ?

		Log in to your WordPress Administration Panel (Dashboard).
		Go to dashboard and navigate to Appearance > Widgets > Drag Image Widgets to Advertise Area Sidebar

	- Navigate to Dashboard > Reading > Click on ( A static page ) from Your homepage displays
 	- Select Homepage as Home and Postpage as Blog

 	- How to setup widgets?

		Navigate to Dashboard > Appearance > Widgets > Drag and Drop T:Spiral custom widgets to BusinessDeal Template Main Section
		Set up pages and posts from the selected widgets
